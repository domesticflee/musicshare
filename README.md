把你的歌单分享到社区
MixShare 音乐歌单分享社区
一款基于 Node.js 搭建的本地歌单分享工具，支持发布、浏览音乐歌单，打造简易音乐分享社区。

🚀 快速启动
环境依赖
确保本地已安装 Node.js 运行环境
启动步骤
进入项目根目录 cd ~\music share
执行启动命令：
bash
运行
node server.js
控制台输出以下内容即启动成功：
plaintext
MixShare running at http://127.0.0.1:8090
浏览器访问地址：http://127.0.0.1:8090
✨ 核心功能
个人歌单编辑、发布至社区
在线浏览所有用户分享的音乐歌单
轻量化部署，零复杂配置，开箱即用
📁 项目目录
plaintext
music share/
├── server.js    # 服务主启动文件
└── README.md    # 项目说明文档
📖 使用教程
按照上方步骤启动本地服务
打开浏览器进入项目主页
编辑你的个人音乐歌单并提交分享
浏览社区内其他用户分享的歌单
